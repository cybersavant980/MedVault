# MedVault -- Offline Home Medical Record & Emergency Management System

MedVault is a **Java Swing-based offline desktop application** developed
to organize and manage personal medical records without requiring an
internet connection or external database.

## Features

-   Patient Management
-   Medicine Management
-   Emergency Management
-   Health Passport
-   Patient-wise PDF Report Upload, View & Delete
-   Interactive Digital Twin
-   Medicine Conflict Detection
-   Offline Data Storage using Java Serialization

## Tech Stack

-   Java
-   Java Swing
-   Java Collections Framework
-   Java IO
-   Object Serialization
-   HashMap, Trie, Queue, Priority Queue

## Local Storage

Application data is stored using Java Object Serialization, while
uploaded PDF reports are organized in:

``` text
data/
└── patient_reports/
    ├── 101/
    ├── 102/
    └── ...
```
